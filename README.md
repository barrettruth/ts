# type

Minimal typing trainer for QWERTY, Dvorak, Colemak-DH, and my custom keyboard
layout "Baremak".

> [!NOTE]
> Due to GitHub's historic unreliability, active development is hosted on
> [Forgejo](https://forge.barrettruth.com/barrettruth/type).
> GitHub is maintained as a read-only mirror.

## Levels

| Level | Name | How it is made |
| --- | --- | --- |
| 01 | home | Generated 3-6 letter tokens from the active layout's home-row letters, mixed with eligible SCOWL words when available. |
| 02 | top | Generated 3-6 letter tokens from home + top-row letters, mixed evenly with eligible SCOWL words when available. |
| 03 | bottom | Generated 3-6 letter tokens from home + bottom-row letters, mixed evenly with eligible SCOWL words when available. |
| 04 | short | Real lowercase SCOWL words shorter than five letters, shuffled on refresh. |
| 05 | prose | Real lowercase SCOWL words, shuffled on refresh. |
| 06 | caps | SCOWL words with a 75% lowercase, 12.5% title-case, 12.5% SCOWL abbreviation mix. |
| 07 | punct | SCOWL word runs with probabilistic prose punctuation and quote spans. |
| 08 | numbers | Generated numeric phrases such as dates, times, counts, versions, measurements, ports, and line references. |
| 09 | symbols | Uniform random symbol tokens from the active layout's symbol set, length 1-5. |
| 10 | sentences | Public-domain Gutenberg sentences, statically filtered and shuffled on refresh. |
| 11 | code | Short placeholder code snippets; full multiline code corpus mode is planned separately. |

The static corpus is generated by `scripts/update-corpus.sh` during `just build`.
Runtime lessons are generated fresh on refresh, layout changes, and level changes.

## Baremak

My custom layout "Baremak" keeps Colemak-DH letters and uses a custom symbol layer.

```text
base
q w f p b   j l u y ;
a r s t g   m n e i o '
z x c d v   k h , . /
```

```text
layer
! @ # < /   \ > & $ |
[ { = ( +   - ) ~ } ] `
% ^ * _ ?   : ; ' "
```
